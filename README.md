# **Dummy Registration System**

A website course registration and scheduling system designed for HTU University. The system includes two main user views: **Student View** and **Employee/Admin View**. Students can browse courses and set schedule preferences, while employees can monitor participation trends and recommended scheduling.

---

## ** Project Overview**

This project simulates a functional **dummy registration portal** where students can select preferred courses and schedules, and employees can view analytics and generate recommended schedules.
